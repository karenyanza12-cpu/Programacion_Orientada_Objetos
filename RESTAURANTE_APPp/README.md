# Sistema de Gestión de Restaurante

## Estudiante

Karen Anahi Yanza Maza

## Descripción

Este proyecto consiste en un sistema básico de gestión de un restaurante desarrollado en Python mediante Programación Orientada a Objetos. El programa permite registrar productos y clientes, almacenarlos en listas, mostrar la información registrada y realizar una venta sencilla que actualiza el stock del producto.

## Estructura del proyecto

```
restaurante_app/
│
├── modelos/
│   ├── __init__.py
│   ├── producto.py
│   └── cliente.py
│
├── servicios/
│   ├── __init__.py
│   └── restaurante.py
│
└── main.py
```

## Tipos de datos utilizados

- **str:** nombres, cédulas y nombre del restaurante.
- **int:** código, edad, stock y puntos.
- **float:** precio del producto.
- **bool:** disponibilidad del producto y estado del cliente.
- **list:** almacenamiento de productos y clientes.

## Reflexión

La Programación Orientada a Objetos permite organizar el código de forma clara mediante clases y objetos. El uso de identificadores descriptivos, tipos de datos adecuados y listas facilita el desarrollo de programas más ordenados, comprensibles y fáciles de mantener.
